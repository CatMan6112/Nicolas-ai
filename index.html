What Is Nicolas AI?

Nicolas AI is an organization deicated to providing kids 17 and under with the resources and guidance to create their own websites and digital projects.

We have state-of-the-art web builders that teach children how to code. We utilize a teaching method known as "Gameified Learning". This is the same method used by many other learning platforms (Such as "Duolingo").

Abuse and Games Policy

  • Using Nicolas AI

    Nicolas AI is meant to be a tool to help and benefiet learning. Any projects that violate this usage, will be immediatly terminated.

  • Alternate Accounts

    Whenever a project is terminated, a copy of the project files are saved and compressed. New files are immediatly crossed reference with known and stored game data to prevent learning disruption.

  • Contact Us or Report a Project

    Nicolas AI strictly prohibits resources being used for the disruption of learning. If a project is using these resources to disrupt learning, it is reccomended to contact Nicolas AI.
